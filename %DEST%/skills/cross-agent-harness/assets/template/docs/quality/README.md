# 文档质量与维护规则

## 目标

本目录用于说明 harness 文档结构的质量要求、当前机械校验覆盖范围和后续待补项。

## 当前校验覆盖

- 必需入口文件与目录存在性。
- `AGENTS.md` 中关键协议入口存在。
- 关键文本文件可按 UTF-8 读取。

## 当前未覆盖范围

- 基于 Git diff 的跨模块文档遗漏检查。
- 正式设计文档与导航页的全量覆盖一致性检查。
- 执行计划字段的语义完整性检查。

## 维护规则

- 结构变化后同步更新 `scripts/validate_harness.py`。
- 新增正式设计文档后同步更新 `docs/design-docs/index.md`。
- 发现入口、命令或索引失效时，优先修正文档与脚本，而不是在聊天中补口头说明。
